# Julián Fernández Mejía - Academic Website

Personal academic website built with the [Academic Pages](https://github.com/academicpages/academicpages.github.io) Jekyll template.

## Pages

- **Home** - About, contact info, news, publications
- **Vitae** - Education, academic positions, industry experience
- **Research** - Published articles, working papers, research in progress
- **Teaching** - Current and previous courses
  - Financial Theory 2025
  - International Finance 2025
  - Office Hours
- **Useful Links** - Programming resources, literature review, data sources

## Running locally

1. Install Ruby, Bundler, and Node.js
2. Run `bundle install`
3. Run `bundle exec jekyll serve -l -H localhost`
4. Visit `http://localhost:4000`
